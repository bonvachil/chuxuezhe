## 一、IP被墙的检测方法

### 方法1：通过KiwiVM控制面板检测
1. 登录搬瓦工官网控制面板
2. 进入Services > My Services查看VPS信息
3. 点击KiwiVM Control Panel进入管理界面
4. 访问检测页面进行测试：
   - 显示"IP NOT BLOCKED"表示正常
   - 显示"IP BLOCKED"则确认被墙

### 方法2：使用Ping.pe工具检测
1. 访问Ping.pe网站
2. 输入您的IP地址和常用端口（如80、443等）
3. 分析检测结果：
   - 国外节点正常而国内不通 → IP被限制
   - 国内外均不通 → 可能是服务器故障
   - 全部正常 → 检查其他网络问题
